# FOSSEE eSim Tasks

This repository contains KiCad project files used for the FOSSEE eSim tasks.

## Contents

- `FOSSEE_TEST/` – KiCad PCB, schematic, project, and backup files for the FOSSEE eSim task.

## How to open

1. Install KiCad.
2. Open `FOSSEE_TEST/FOSSEE_TEST.kicad_pro` in KiCad.
