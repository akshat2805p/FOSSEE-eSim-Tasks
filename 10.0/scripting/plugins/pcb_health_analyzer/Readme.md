# eSim KiCad Health Analyzer

## Objective

A KiCad Action Plugin developed for eSim/FOSSEE workflows to analyze PCB design quality and generate engineering health reports.

## Features

- PCB statistics analysis
- Via and track inspection
- Net analysis
- JSON report generation
- Modular plugin architecture

## Technologies Used

- Python
- KiCad pcbnew API
- JSON
- GitHub

## Future Scope

- PCB health score system
- DRC enhancement
- GUI dashboard
- AI-assisted PCB suggestions

## New Features Added

- Layer-wise PCB statistics
- Via and drill analysis
- Automatic PCB health report generation
- Improved formatted console output
- KiCad Action Plugin integration
